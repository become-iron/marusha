Приложение создано с целью собрать в одном месте лучшие фичи различных других приложений для изучения японских азбук хираганы и катаканы. Прогресс, набранный в ходе тренировок, сохраняется.

Источники "вдохновения":

  • Тренажёр для заучивания хираганы/катаканы
  • hiragana.training
  • Android-приложение "Hiragana - Learn Japanese"
  • Android-приложение "JVocab - Японский язык"

Репозиторий приложения: github.com/become-iron/marusha
